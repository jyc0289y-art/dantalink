## 3. 사용 흐름 (사용자 인계)

```
1. https://jyc0289y-art.github.io/dantalink/docs/data_collector_gui.html
2. (모든 필드 빈 상태 그대로) ▶ INITIATE SCAN
3. 우측 패널 상단에 "▸ Trending Keywords (Top 12)" 섹션 자동 표시
4. 관심 chip(예: "이란", "백신") 클릭 → 황색 테두리로 선택 표시
5. "+ Add Selected to Watch" 클릭 → 좌측 Watch Keywords로 이동
6. 다시 ▶ INITIATE SCAN → 선택 키워드 관련 이벤트만 좁혀짐
7. ▶ FORWARD TO MATCHER → Stock Match로 전달
```
