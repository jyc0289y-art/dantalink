## 4. 미해결 / 유의

- **TradingView 위젯 한계**: 일부 코스닥 종목은 차트 데이터 부재 가능 (위젯 자체는 로드되지만 "no data" 표시 가능)
- **iframe 임베드 차단 정책**: 네이버/다음 금융 모두 X-Frame-Options 차단 → iframe 임베드 불가능 (그래서 새 탭 링크 채택)
- **링크 검증**: HEAD 요청만 검증 (실 콘텐츠 정상 여부는 사용자 브라우저에서 확인)

```
1. Stock Match 종목 카드 클릭

5. 📰 뉴스 → 클릭으로 새 탭 (네이버 뉴스/Google 뉴스 등)
6. 💬 종목토론 → 클릭으로 새 탭 (네이버 종목토론/팍스넷)
7. 닫기: ✕ / 외부 클릭 / Esc
```
