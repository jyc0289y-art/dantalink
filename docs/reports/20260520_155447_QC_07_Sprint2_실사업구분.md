## 4. 알고리즘 한계 / 향후 개선

- **단순 substring 매칭**: "로보틱스" 키워드 vs "로봇" 사업 — 미스
  - 개선: 키워드 동의어 사전 (로보틱스↔로봇, AI↔인공지능 등)
- **사피엔반도체 GENUINE 판정**: Industry "반도체"는 맞지만 실제는 DDI. "메모리 슈퍼사이클" 모멘텀과는 무관
  - 현재는 키워드 단위 cross-check만. 향후: 키워드와 사업의 의미 거리 점수화 (LLM 임베딩)
- **B fallback 수동 사전 미작성**: 현재 C(자동)만 작동. fdr가 안정적이라 fallback 불필요했음. 필요 시 향후 추가.
