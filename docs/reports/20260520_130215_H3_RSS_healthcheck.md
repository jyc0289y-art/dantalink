**문제 피드 상세**:

### Reuters (도메인 소멸 확인)
```
ERROR: <urlopen error [Errno 11001] getaddrinfo failed>
```
DNS 단계에서 `feeds.reuters.com` 호스트를 찾을 수 없음. Reuters Agency가 RSS 서비스를 종료한 것으로 보임 (2020년대 중반 알려진 변경). 대안 없음 → GDELT가 Reuters 기사를 인덱싱하므로 그쪽으로 흡수.

### 한국경제 (적극 차단)
```
status=301 redirect, then BOZO=<unknown>:69:50: undefined entity
```
다른 URL 후보 3건 모두 `403 Forbidden` 또는 동일한 entity 오류 → 한국경제가 RSS 외부 접근을 차단/봉쇄한 것으로 추정. 대체 URL 시도:
- `https://www.hankyung.com/feed/economy` → 403
- `https://www.hankyung.com/feed/all-news` → 403
- `https://www.hankyung.com/feed/finance` → 403

→ 한국경제는 완전 제거. 매일경제·파이낸셜뉴스로 보강.
