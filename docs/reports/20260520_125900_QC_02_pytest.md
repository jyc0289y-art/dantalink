## 3. Deprecation Warning 동작 확인

```python
from dantalink import CollectorConfig
from dantalink.collectors import KRXCollector

cfg = CollectorConfig(keywords=['test'])
# 모듈 import만으로는 warning 미발생 (확인됨)

c = KRXCollector(cfg)
# → [DeprecationWarning] KRXCollector is deprecated; use DARTCollector instead.
#    (원래 이 클래스는 KRX가 아닌 DART API를 호출하므로 이름이 변경됨)

print(type(c).__name__, c.source)
# → KRXCollector EventSource.DART
```

**의미**: KRXCollector는 DARTCollector의 subclass이므로 `c.source`가 자동으로 `EventSource.DART`. 기존 코드를 그대로 두어도 데이터는 `dart` source로 저장됨 (실제 호출 대상과 일치).
