## 4. 시스템 안정성

- **에러 격리**: BaseCollector.safe_fetch + BaseMatcher.safe_match로 단일 실패가 파이프라인을 막지 않음
- **인코딩 호환**: 윈도우 cp949 콘솔에서도 한글·유니코드 정상 출력 (cli.py reconfigure + PYTHONIOENCODING)
- **Rate Limit 대응**: GDELT 5.5초/req, User-Agent 명시 (M2 일부 미리 적용)
- **DB 캐시**: KRX 종목은 CSV에 캐시되어 매 실행마다 재다운로드 불필요
- **JSON UTF-8**: 모든 산출물(events·matches)이 한글 유실 없이 직렬화
