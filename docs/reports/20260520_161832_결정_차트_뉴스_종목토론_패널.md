## 2. 채택안

- **차트**: TradingView 위젯 임베드 (`s.tradingview.com/widgetembed/?symbol=KRX:코드`) — iframe 안정적 알려짐, KRX 종목 지원
- **뉴스 + 종목토론**: **새 탭 링크** (네이버 금융) — iframe 차단 거의 확실, 단타에서 외부 페이지 새 창은 자연스러운 흐름

- 네이버 뉴스: `finance.naver.com/item/news.naver?code={code}`
- 네이버 종목토론: `finance.naver.com/item/board.naver?code={code}`
