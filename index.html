DANTALINK

BEHAVIORAL FINANCE × KOREAN MARKET × SHORT-TERM ALPHA

한국 시장의 음운/카테고리 매칭으로 발생하는 단기 모멘텀을 시스템적으로 포착하는 도구. 카너먼·트버스키의 휴리스틱, Rashes(2001) 이름 유사성 효과, Da/Engelberts/Gao(2011) 주목 기반 거래 등 행동재무학 핵심 이론을 알고리즘으로 정형화. SeouLink 산하 프로젝트.

▸ Quick Launch

💡 권장 흐름: 📡 Data Collection (키워드 + 소스 + 데이터셋 필터) → ▶ FORWARD TO MATCHER → 📈 Stock Match에서 자동 매칭. 두 콘솔은 localStorage로 연계됩니다.

▸ Try it locally

# 설치
git clone https://github.com/jyc0289y-art/dantalink.git
cd dantalink
pip install -e ".[dev]"
pip install finance-datareader

# 라이브 수집 (RSS 단독)
python -m dantalink.cli scan \
    --keywords "AI,에볼라,OpenAI" --timespan 24h \
    --sources rss --max-records 20 --output events.json

# 종목 매칭
python -m dantalink.cli match \
    --input events.json --output matches.json \
    --top 30 --min-marcap 100 --max-marcap 5000 --sort max
